# FatihRobotDevBoard
Fatih Üniversitesi Robot Geliştirme Kartı CCS C Örnek Kodları

Pinler:
Buton         <----> E0
Led1          <----> E1
Led2          <----> E2
Sens0         <----> A0 - GND -VCC
Sens1         <----> A1 - GND -VCC
Sens2         <----> A2 - GND -VCC
Motor1 Yön1   <----> D2
Motor1 Yön2   <----> D3
Motor2 Yön1   <----> D0
Motor2 Yön2   <----> D1
Motor1 Enable <----> C2 (CCP2)
Motor2 Enable <----> C1 (CCP1)
TX            <----> C6
RX            <----> C7
